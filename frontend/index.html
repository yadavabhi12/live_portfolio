AI-Powered Chat & Real-Time Data System

Personal Full-Stack Project

Oct 2025 · Ongoing

Personal Project · Remote

Built an AI-first conversational assistant integrated into a full-stack productivity platform. The system combines real-time WebSocket data, an LLM-backed personal chatbot, and vector-based retrieval so users get fast, human-like answers about their profile, skills, and projects. Designed to demonstrate practical AI engineering, secure data handling, and smooth frontend UX for technical reviewers and HR.

Primary Capabilities & Highlights:

  • Human-friendly personal chatbot that answers resume- and profile-related questions using contextual retrieval (trained on personal dataset & project docs)
  • Real-time features powered by WebSocket for live chat, notifications, and instant task updates across connected clients
  • Vector search with Pinecone + embedding model (Gemini embeddings) to retrieve relevant resume snippets and project details for accurate replies
  • Hybrid LLM orchestration using Gemini & OpenAI models with LangChain-style pipelines and local retrieval augmentation (RAG) for factual, context-aware responses
  • Integration with third-party APIs (Tavily/Tavily-like) for enriched conversational prompts and real-time data fetching
  • Secure data handling: user-scoped ingestion, scoped embeddings, and optional redaction for private fields before indexing

Technical Details & Flow:

User data (resume, project notes, profile) → preprocessing & embeddings → Pinecone vector store. Conversational requests follow a pipeline: retrieve relevant passages → construct LLM prompt with context → call LLM (Gemini/OpenAI) → post-process and return to client over WebSocket/HTTP. LangChain-style components orchestrate retrieval, prompt templates, and response caching for low-latency replies.

Technologies Used:

React.js Tailwind CSS JavaScript Redux Node.js Express.js WebSocket MongoDB MySQL Pinecone Gemini / OpenAI LangChain-style Tavily API GitHub

Professional Impact:
